Convex Hull Visualizer

Steps

Prepare the points

Find a corner point

Find the point with the lowest x and y coordinate. Since the y axis is inverted when drawing, this means finding the top leftmost point.

Sort points by angle

Create a list of the points by angle in counter-clockwise order.

Remove points with shared angles

Since we want to encapsulate all points, we want to prioritize points that are further out (and ignore closer ones).

Check if there are enough points

If there are exactly 2 other points left, we have a triangle. If there are less, there is no shape. Otherwise, we proceed through the rest of the points.

Traversing through points

Add three points to a 'stack'

Take the first three points from our sorted list and add them to a 'stack'. Use a pseudo-stack since we always want to look at the top three points, and only ever have to remove the second newest point. This stack will eventually form our convex hull.

Determine the angle between the points

Find the angle from the last added point to the second last, and from the second last added point to the third last. We can find the difference between the angles, and determine if it is clockwise (concave) or counterclockwise.

Remove concave points

If the angle between the last added three points was clockwise, we would have a concave angle, meaning the second last added point must be removed from the stack.

Repeat for points in order

Repeat the point addition, angle check, and concave removal for all points in our list or sorted points.
